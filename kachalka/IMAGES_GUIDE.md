# Руководство по добавлению изображений

## Способ 1: Через Django Admin (самый простой)

1. Запустите сервер: `python manage.py runserver`
2. Откройте: http://localhost:8000/admin/
3. Войдите: `admin` / `admin`
4. Перейдите в раздел "Спортивные залы"
5. Выберите зал
6. Прокрутите вниз до секции "Изображения зала"
7. Нажмите "Добавить еще одно Изображение зала"
8. Загрузите изображение (рекомендуется 4-8 изображений на зал)
9. Сохраните

## Способ 2: Скачать бесплатные изображения

### Где скачать изображения залов:
- **Unsplash**: https://unsplash.com/s/photos/gym-interior
- **Pexels**: https://www.pexels.com/search/gym-equipment/
- **Pixabay**: https://pixabay.com/images/search/fitness-gym/

### Рекомендуемые поисковые запросы:
- "gym interior"
- "fitness equipment"
- "weight room"
- "cardio machines"
- "yoga studio"
- "swimming pool gym"

## Способ 3: Использовать placeholder изображения

Если нужны временные изображения для разработки:

1. Создайте папку `temp_images` в корне проекта
2. Положите туда изображения (любые)
3. Загрузите их через админку

## Текущее состояние

- Залы созданы БЕЗ изображений
- Тренеры созданы БЕЗ изображений

## Примечание

Изображения не критичны для функциональности приложения.
Все работает и без них, просто показывается заглушка "Нет фото".
